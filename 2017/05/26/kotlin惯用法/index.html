data class

有时候需要一些类要保存数据，而不需要其他操作，当然基本操作除外，kotlin为data class提供的基本操作有：

  • equals() 相当与“==”操作
  • hashCode() 计算hash值
  • toString() 函数
  • componentsN() 函数
  • copy() 拷贝
