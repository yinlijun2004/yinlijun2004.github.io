发表于 2016-11-28

问题出现环境

  • Ubuntu 12.04
  • Android Studio 2.2.2

解决步骤

开启VT-x

并且提示要在BIOS里面开启VT-x。

重启电脑，按DEL键进入BIOS,发现确实没有启用，于是启用后再此重启电脑。

此时打开Android Studio，仍然提示一样的错误。
